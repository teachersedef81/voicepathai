Voice Command Guide

VoicePathAI understands natural language commands. Here are some examples to get started:

  • 1
    "Go to features section" - Navigates to the features section
  • 2
    "Read the current section" - Reads all text in the current section
  • 3
    "Scroll down" - Scrolls down the page
  • 4
    "Stop reading" - Stops the current speech output
  • 5
    "Help" - Repeats these voice command options

Tip: Say "What can I say?" at any time to hear available commands.
